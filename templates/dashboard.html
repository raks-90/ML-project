Dashboard Overview

This AI Dashboard uses Deep Learning and Artificial Intelligence to classify fingerprint images as REAL or FAKE.

Total Predictions

Displays total fingerprint images processed by the system. total=real+fake

Accuracy

Accuracy shows how correctly the model predicts results.

Accuracy = (Correct Predictions / Total Predictions) × 100

FAR (False Acceptance Rate)

FAR measures fake fingerprints incorrectly accepted as REAL.

FAR = (False Accepted / Total Fake Attempts) × 100

FRR (False Rejection Rate)

FRR measures genuine fingerprints incorrectly rejected.

FRR = (False Rejected / Genuine Attempts) × 100

Weekly Prediction Graph

Displays weekly REAL and FAKE prediction analytics.

FAR vs FRR Graph

Compares system security and reliability using line graph analytics.

Download Option

Allows users to download prediction reports and dashboard analytics.

Reset Option

Clears dashboard data, graphs and prediction records.

Send Option

Sends prediction results and reports directly.
